Window obyekti


Javascript daxilindəki bu obyekt brauzerdəki açılmış pəncərə ilə işləmək üçün nəzərdə tutulur. Bu obyekt vasitəsilə mövcud pəncərə ilə işləmək və onun haqqında məlumatları əldə etmək mümkündür. Məsələn, window obyektindəki close() funksiyasını işlədərək açıq olan pəncərəni bağlaya bilərik 

                    
                function bagla(){
                window.close();
                }

Nəticədə düyməyə basdıqda işlədiyimiz pəncərə bağlanmış olacaq.

Open funksiyası


Mövcud pəncərəni bağlamaqdan əlavə yeni bir pəncərə də aça bilərik. Bunun üçün open funksiyasından istifadə edə bilərik. 

          function yeni(){
          window.open("http://google.az");
          }
